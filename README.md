# YouTube-Transcript-Summarizer
YouTube Transcript Summarizer made with Python NLP and YouTube API.

Input the URL in the GUI Input box, and on the process button. If the closed captions aren't disabled by the uploader of the target video, the YouTube Transcript API fetches the transcripts in JSON format, which is then transformed into Human understandable language by the Transformers library's HUggingface module.
